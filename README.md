# GymHQ Landing Page

A minimalist landing page for GymHQ - a comprehensive gym management platform for gym owners to manage memberships, cafes, and everything in between.

## About GymHQ

GymHQ is a complete gym management platform designed to simplify gym operations. This landing page serves as the official web presence for WhatsApp API verification with Meta.

## Getting started

To get started, first install the npm dependencies:

```bash
pnpm install
```

Next, run the development server:

```bash
pnpm dev
```

Finally, open [http://localhost:3000](http://localhost:3000) in your browser to view the website.

## Features

- Minimalist, clean design with light theme
- Figtree font family
- Dark zinc color scheme
- Simple "Book Demo" CTA with mailto link to karn@gymhq.in
- Responsive design
- Built with Next.js 15 and Tailwind CSS 4

## Tech Stack

- [Next.js](https://nextjs.org) - React framework
- [Tailwind CSS](https://tailwindcss.com) - Utility-first CSS framework
- [TypeScript](https://www.typescriptlang.org) - Type safety
- [Figtree](https://fonts.google.com/specimen/Figtree) - Font family
